開機程式注意事項

M-220

• M-220開機程式分為兩種版本：1.04E 及 2.00E

• M-220機體序號的末四碼：
7370以前是 1.04E
(不能正常開機時會出現 Loading Now…)
7371以後是 2.00E
(不能正常開機時會出現 Insert System Disk)

• 使用不正確的開機程式會出現錯誤：
( *System Error* ) or ( Insert System Disk )

• 請注意：M-220 所有開機程式及 NC 程式都必須存放在根目錄

• 下載後，解開程式再複製到磁片使用

M-430

• 下載後，解開程式再複製到CF卡使用

• 請注意：M-430 所有開機程式及 NC 程式都必須存放在根目錄

• 點擊下載：1.02E

P-530

• 下載後，解開程式再複製到USB使用

• 請注意：P-530開機程式必須存放在根目錄下，NC程式可儲存於資料夾內

• 點擊下載：1.02E
